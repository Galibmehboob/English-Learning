## 🚀 English Window – Vocabulary Learning Web App

English Window is an interactive vocabulary learning web application where users can explore words through different lesson levels. Each word includes **audio pronunciation**, along with a detailed information section showing its **meaning, example sentence, and synonyms**.

The platform also includes a **search feature** that allows users to quickly find words, making it work like a mini dictionary.

This project helped me improve my understanding of **JavaScript DOM manipulation, search filtering, interactive UI design, and audio-based features**.

The project is still under development, and more features will be added in the future 🚀
